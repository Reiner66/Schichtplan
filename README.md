# 🦊 SchichtPlan – CGM Hüttenfuchs Edition

**Persönlicher Schichtkalender für Schichtarbeiter** – als installierbare PWA für Handy, Tablet und PC.

👉 **[App öffnen](https://reiner66.github.io/Schichtplan/)**

---

## 📱 App installieren

**Android** (Chrome): Menü ⋮ → „Zum Startbildschirm hinzufügen"  
**iPhone** (Safari): Teilen ↑ → „Zum Home-Bildschirm"

---

## ✨ Funktionen

- 📅 Persönlicher Schichtverlauf per Texteingabe (F/S/N/B/-)
- 🔄 Mehrere Planabschnitte (z.B. Schichtwechsel ab einem Datum)
- 🏖 Urlaub, Z-Tage, V-Tage, N-Freischicht, Sonderurlaub
- 📊 Jahresstatistik: Ist-Stunden, Z/V-Tage, Resturlaub, N-Freischichten
- ⚠️ AZK-Warnung: Z-Tage zuerst verbrauchen (wandern sonst am 31.12. ins AZK)
- 🌙 N-Freischicht nach Stahlnovelle (÷17 oder pauschal 4/Jahr)
- ❄️ Winterurlaubstag automatisch berechnet
- 🎉 Persönliche Termine (einmalig oder jährlich wiederkehrend)
- 🗺 Feiertage + Schulferien für alle 16 Bundesländer
- 🎨 4 Design-Themes (Midnight / Dusk / Slate / Day)
- 💾 Export & Import aller Daten als JSON
- 🔒 Alle Daten bleiben lokal – nichts wird übertragen!
- 📲 Offline-fähig (Service Worker)

---

## 🔒 Datenschutz

**Alle Daten werden ausschließlich lokal im Browser gespeichert.**  
Es findet keine Übertragung an Server statt. Niemand außer dir kann deine Daten einsehen.
